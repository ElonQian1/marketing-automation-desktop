# ADB 领域核心层

## 职责
- 定义ADB领域模型和实体
- 实现核心业务规则
- 领域服务和领域逻辑
- 领域事件定义

## 架构原则
- 不依赖外部框架和技术细节
- 包含核心业务规则和约束
- 定义领域模型的行为和状态
- 领域服务处理复杂业务逻辑

## 文件组织
```
domain/
├── index.ts              # 领域层统一导出
├── entities/             # 领域实体
│   ├── Device.ts         # 设备实体
│   └── Connection.ts     # 连接实体
├── valueObjects/         # 值对象
├── services/            # 领域服务
└── events/              # 领域事件
```
