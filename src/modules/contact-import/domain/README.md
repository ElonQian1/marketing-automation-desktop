# 联系人导入 领域核心层

## 职责
- 定义联系人领域模型和实体
- 实现导入业务规则和约束
- 联系人数据验证和处理规则
- 导入策略和算法逻辑

## 架构原则
- 不依赖外部技术框架
- 包含核心业务规则和约束
- 定义联系人模型的行为和状态
- 导入策略的业务逻辑

## 文件组织
```
domain/
├── index.ts              # 领域层统一导出
├── entities/             # 领域实体
│   ├── Contact.ts        # 联系人实体
│   └── ImportSession.ts  # 导入会话实体
├── valueObjects/         # 值对象
├── services/            # 领域服务
└── events/              # 领域事件
```
