# 精准获客 领域核心层

## 职责
- 定义客户和获客领域模型
- 实现获客业务规则和策略
- 客户画像和分析算法
- 获客任务和目标管理

## 架构原则
- 不依赖外部技术框架
- 包含核心获客业务规则
- 定义客户模型的行为和属性
- 获客策略的业务逻辑

## 文件组织
```
domain/
├── index.ts              # 领域层统一导出
├── entities/             # 领域实体
│   ├── Customer.ts       # 客户实体
│   ├── Task.ts          # 获客任务实体
│   └── Target.ts        # 获客目标实体
├── valueObjects/         # 值对象
├── services/            # 领域服务
└── events/              # 领域事件
```
