员工管理GUI程序使用说明
======================

安装和使用：
1. 解压所有文件到任意文件夹
2. 双击运行 employee-gui.exe

系统要求：
- Windows 10/11 (64位)
- 如需使用Android设备功能，请确保：
  * 已安装Android设备驱动
  * 设备已开启USB调试模式
  * 设备已通过USB连接到电脑

目录结构：
employee-gui.exe       - 主程序
platform-tools/        - Android调试工具(ADB)
README.txt             - 本说明文件

注意事项：
- 首次运行可能需要"以管理员身份运行"
- 请勿删除 platform-tools 文件夹，程序依赖其中的工具
- 如遇到杀毒软件误报，请添加程序到白名单
