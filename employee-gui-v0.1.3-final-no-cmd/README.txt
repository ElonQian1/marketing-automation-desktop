员工管理GUI程序使用说明
======================

安装和使用：
1. 解压所有文件到任意文件夹
2. 双击运行 employee-gui.exe

系统要求：
- Windows 10/11 (64位)
- 如需使用Android设备功能，请确保：
  * 已安装Android设备驱动
  * 设备已开启USB调试模式
  * 设备已通过USB连接到电脑

目录结构：
employee-gui.exe       - 主程序
platform-tools/        - Android调试工具(ADB)
README.txt             - 本说明文件

故障排除：
- 如果出现"无法访问此页面"或"ERR_CONNECTION_REFUSED"错误：
  1. 右键点击 employee-gui.exe
  2. 选择"以管理员身份运行"
  3. 如仍有问题，请检查防火墙设置
  
- 如果杀毒软件误报：
  1. 将整个程序文件夹添加到杀毒软件白名单
  2. 或暂时关闭实时防护后运行

注意事项：
- 首次运行建议"以管理员身份运行"
- 请勿删除 platform-tools 文件夹，程序依赖其中的工具
- 如果程序无法启动，请确保所有文件都已正确解压
