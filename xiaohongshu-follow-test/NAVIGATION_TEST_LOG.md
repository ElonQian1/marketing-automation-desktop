## 成功的导航步骤

### 步骤1: 主页 -> 侧边栏
```bash
# 检查主页状态
cargo run -- debug --device emulator-5554
# 结果: MainPage, 信心度0.80, 关键元素["主页导航"]

# 点击左上角头像
adb -s emulator-5554 shell input tap 60 100

# 验证侧边栏打开
cargo run -- debug --device emulator-5554
# 结果: SidebarOpen, 信心度0.90, 关键元素["主页导航", "侧边栏菜单", "发现好友页面"]
```
